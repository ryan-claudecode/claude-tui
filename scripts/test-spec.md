# Test Spec — Single Task

Do exactly ONE thing:

1. Open `src/components/StatusBar.tsx`
2. Add a clock that shows current time (HH:MM:SS) on the right side, updating every second
3. Build: `npx electron-vite build`
4. If build passes, commit: `git add -A && git commit -m "feat: add clock to status bar"`
5. Push: `git push`
6. Report what you did, then exit.

That's it. One task. Build, commit, push, exit.
